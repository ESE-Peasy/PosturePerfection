Detailed Description

This class allows for navigation around the application from the settings page.

This class lets the user navigate to the mainwindow, settingspage and datapage from the fullscreen page.

This class handles: Navigating to the mainwindow. Navigating to the settingspage. Navigating to the datapage. Triggering posture reseting functionality
