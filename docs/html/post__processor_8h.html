typedef struct PostProcessing::ProcessedResults PostProcessing::ProcessedResults

The PostProcessing::Coordinate of each body part.

Each body part is expressed as a PostProcessing::Coordinate to indicate the best estimate for that body part's location in the current frame. This is not necessarily the same as the output of the pose estimation model for a given frame.

Settings for an IIR filter.

Wraps the second-order section coefficients for an IIR filter. The sos_coefficients are a 2D array, where the rows correspond to each of the second order stages, and the columns correspond to coefficients for each stage.
